# 欢迎来到 ximu

## 关于 ximu

欢迎使用 ximu —— 全球领先的AI驱动品牌影响力分析平台。在生成式AI日益成为用户获取信息首要入口的今天，ximu让您能够量化、监测并优化您的品牌在AI世界中的形象与地位。

我们的使命： 帮助您量化并掌控您在AI世界中的品牌影响力，让您的品牌在AI搜索中被优先推荐，被准确理解，并始终在竞争中保持领先。

## ximu 是什么

ximu通过持续监控和分析主流AI模型（如GPT-4o-mini等）对用户问题的回答，为您提供关于品牌表现的深度数据洞察。我们如同您的AI世界雷达，帮您解答以下核心问题：

### 核心问题

- 我的品牌在AI回答中被提及的频率和位置如何？
- AI是如何评价我的品牌的？情绪是正面还是负面？
- 我的品牌故事被哪些网站来源所塑造？
- 与竞争对手相比，我的品牌在AI眼中的影响力排名第几？
- 用户在询问哪些与我的行业相关的问题？

# 快速入门

## 快速入门说明

只需短短几分钟设置，ximu即可开始为您工作。

访问 ximu 官网。

## 配置您的品牌与产品

- 主品牌名称： 例如：ximu  
- 产品名称： 例如：GEO分析平台

7天后，您将收到通知。再次登录即可解锁您的专属品牌影响力仪表板，并享受7天全功能免费试用。

# 理解您的核心指标

## 指标说明

ximu使用一套标准化的AI品牌影响力指数来衡量品牌表现。

### 指标表格

| 指标 | 定义与原理 | 为何重要 |
|------|------------|----------|
| 品牌影响力 | 可见度 × 情绪的综合得分，是判断品牌整体健康度的终极指标。 | 一个高影响力的品牌，既要有声量，也要有好口碑。 |
| AI可见度 | 综合提及率、排名和内容占比计算得出的品牌整体曝光强度分数。 | 反映品牌在AI世界中的总体声量大小。 |
| 提及率 | AI回答中提到您品牌的频率。 | 类似于AI世界的“广告牌曝光次数”，频率越高越好。 |
| 提及顺序 | 品牌在AI回答中出现位置的绝对排名。数字越小代表越靠前。 | 位置越靠前，被用户注意和记住的可能性越大。 |
| 内容占比 | 在提及您品牌的回答中，关于您的内容所占篇幅比例。 | 展示品牌叙述的深度，而非仅被点名。 |
| 情绪倾向 | AI回答中传递的情绪，范围 -1（负面）到 +1（正面）。 | 衡量品牌声誉，反映“情绪温度”。 |

# 常见问题解答

## FAQ

### Q1: 平台数据多久更新一次？
A: 系统每日更新品牌分析数据。

### Q2: 平台使用哪些AI模型？
A: 当前核心模型为 GPT-4o-mini，未来将加入更多大模型（Gemini、Claude等）。

### Q3: 数据来源自哪些地区？
A: 新加坡、中国大陆与台湾，并持续扩展更多国家。

### Q4: 我需要自己提供内容吗？
A: 您只需提供品牌、产品及竞品信息。未来将推出自定义查询功能。

### Q5: 为什么某些竞品会缺失数据？
A: 若AI回答未提及某竞品，则该日数据可能为空值。
