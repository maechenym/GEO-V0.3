# 文档编写模板 - 简化版

## 📝 模板规则

### 一级标题（H1）`#` → 侧边栏主分类（带图标）
- **格式**: `# 标题名称`
- **显示**: 侧边栏大标题，带图标
- **示例**: `# 快速入门`、`# 核心功能模块详解`

### 二级标题（H2）`##` → 侧边栏可点击项
- **格式**: `## 标题名称`
- **显示**: 侧边栏子项目，可点击跳转
- **示例**: `## 注册账号`、`## 总览（Overview）`

### 三级标题（H3）`###` → 仅文档内容（不在侧边栏）
- **格式**: `### 标题名称`
- **显示**: 仅出现在文档内容中
- **示例**: `### 核心KPI指标卡`

---

## 📄 文档结构模板

```mdx
---
title: 文档标题
description: 文档描述
---

# 主分类1（H1 - 侧边栏显示，带图标）

主分类说明...

## 子章节1（H2 - 侧边栏可点击）

子章节内容...

### 详细说明（H3 - 仅文档内容）

详细内容...

## 子章节2（H2 - 侧边栏可点击）

子章节内容...

---

# 主分类2（H1 - 侧边栏显示，带图标）

主分类说明...

## 子章节1（H2 - 侧边栏可点击）

内容...
```

---

## ✅ 编写步骤

1. **将你的文档内容放入 `content/docs/index.mdx`**
2. **按照模板规则使用标题层级**：
   - `#` 用于主分类
   - `##` 用于可点击的子章节
   - `###` 用于文档内容细分
3. **更新 `src/lib/docs/navigation.ts`** 中的导航配置，匹配你的文档结构

---

## 📋 识别规则总结

| Markdown | 侧边栏显示 | 说明 |
|----------|-----------|------|
| `# 标题` | ✅ 主分类（大标题+图标） | 需要手动在 navigation.ts 中配置 |
| `## 标题` | ✅ 子章节（可点击） | 需要手动在 navigation.ts 中配置 |
| `### 标题` | ❌ 仅文档内容 | 自动生成锚点，但不在侧边栏 |

---

## 💡 重要提示

- **导航配置需要手动维护**：在 `src/lib/docs/navigation.ts` 中配置侧边栏结构
- **文档内容自动渲染**：MDX 文件会自动渲染到页面
- **锚点自动生成**：所有标题都会自动生成锚点 ID

按照这个模板编写文档，然后在 `navigation.ts` 中配置对应的导航结构即可！
