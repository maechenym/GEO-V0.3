## 🚀 下一步：部署到 Vercel

### 步骤 1: 访问 Vercel

1. 打开 https://vercel.com
2. 点击 **"Sign Up"** 或 **"Login"**

### 步骤 2: 使用 GitHub 登录

- 选择 **"Continue with GitHub"**
- 授权 Vercel 访问你的 GitHub 账号

### 步骤 3: 导入项目

1. 点击 **"Add New Project"** 或 **"Import Project"**
2. 在项目列表中找到 **GEO-V0.3**
3. 点击 **"Import"**

### 步骤 4: 配置项目

- Framework Preset: **Next.js**（自动检测）
- Root Directory: `./`（默认）
- Build Command: `npm run build`（默认）
- Output Directory: `.next`（默认）

### 步骤 5: 添加环境变量 ⚠️ 重要！

1. 在 **Environment Variables** 部分
2. 点击 **"Add"**
3. 变量名：`NEXT_PUBLIC_USE_MOCK`
4. 值：`true`
5. 点击 **"Add"**

### 步骤 6: 部署

1. 点击 **"Deploy"** 按钮
2. 等待 2-3 分钟构建完成

### 步骤 7: 获得链接 🎉

- 部署完成后，你会看到一个链接
- 格式：`https://geo-v0-3.vercel.app` 或类似
- **这个链接可以分享给任何人！**
